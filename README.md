# myhome

집 평면도 기반 3D 가구 배치 프로토타입입니다.

## 주요 기능
- 3D 화면에서 평면도(73㎡ 확장형 기본 가정) 확인
- 의자/책상/침대/소파 선택 후 바닥 클릭으로 배치
- 배치된 가구 드래그 이동

## 실행 방법
별도 빌드 없이 정적 파일로 동작합니다.

```bash
cd /home/runner/work/myhome/myhome
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000` 접속 후 사용합니다.

## 평면도 반영 범위
요청된 링크 기반 분석 정보는 실행 환경에서 외부 도메인 접근 제한으로 직접 조회가 불가능하여,
73㎡ 타입의 확장형 기본 구성(거실+주방 공용공간 중심, 침실 분리 구조)을 기본 레이아웃으로 반영했습니다.
